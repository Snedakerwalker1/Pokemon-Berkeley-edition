Pokemon Berkeley Edition

made by Walker Snedaker



In this file you will find a fully playable game based on the pokemon franchise and the uc berkeley campus. I built this game usign the Pokemon Essentials, found at https://essentialsdocs.fandom.com/wiki/Essentials_Docs_Wiki, package for the RPG maker system.

In implementig this project I learned a ton of photoshop techniques to make some new map animations and a ton of character sprites as well as how to use the Ruby coding language to skript some of the intricat game mechancis used in the game.

If you want to play this game feel free to downlad the repository and open the .exe file to start and play the game.
